# Apresentação do Projeto

Este projeto consiste no desenvolvimento de um **Sistema de Informação** para uma **Empresa de Transporte Aéreo**, elaborado no contexto da disciplina de **Algoritmos e Estruturas de Dados II** da UFMT.

O sistema permite que usuários autenticados realizem operações de **cadastro, listagem, pesquisa, edição e exclusão de voos**, com persistência dos dados em arquivos CSV e registro de todas as operações em um arquivo de auditoria (logs).

A entidade principal do sistema é o **voo**, identificada por uma chave única (ID), contendo informações como código do voo, origem e destino. O sistema foi desenvolvido **exclusivamente em linguagem C**, utilizando uma arquitetura modular baseada em **Tipos Abstratos de Dados (TADs)** e permitindo a execução das interações por meio de arquivos de entrada `.txt`.

O projeto tem como objetivo aplicar, de forma prática, os conceitos de estruturas de dados, manipulação de arquivos, controle de acesso e boas práticas de programação.
